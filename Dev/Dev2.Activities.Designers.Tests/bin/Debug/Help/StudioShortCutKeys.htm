Studio Short Cut Keys

A fully comprehensive look at short cut keys in the Studio.

Alt

Generally we use Alt for opening or focusing on areas of the system like the Connection Manager area, or the New Workflow wizard. Typically if that area is already open, then the shortcut will bring that area into focus in the Work Environment.

Alt + F → Feedback window

Alt + S → Scheduler

Alt + T → Settings

Alt + F4 → Exit Studio

F

Generally we use function keys for execution of a workflow.

F5 → Show Inputs screen

F6 → Debug workflow with saved inputs

F7 → View workflow output in browser with saved inputs

Ctrl

Ctrl is typically used for executing tasks like Save, Cut, Undo etc. The Ctrl short cuts are context based. For example, if the context of the page is Settings, then Ctrl + R (Run or Execute) is meaningless and will do nothing.

Ctrl + S → Save

Ctrl + R → Run (Execute)

Ctrl + D → Deploy

Ctrl + T → Debug Test

Ctrl + X → Cut

Ctrl + C → Copy

Ctrl + V → Paste

Ctrl + Z → Undo

Ctrl + Y → Redo

Ctrl + W → New Workflow

Ctrl + Shift

Ctrl + Shift + D → New Database Service

Ctrl + Shift + P → New Plugin Service

Ctrl + Shift + W → New Web Service
