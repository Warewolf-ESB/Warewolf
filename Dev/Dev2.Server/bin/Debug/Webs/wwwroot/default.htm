Warewolf Data Return Types

You can return data from Warewolf in either XML or JSON by supplying a type to the end of your URL.
If you do not provide one the default return type is XML.

  • If you have a service at the following URL http://myServer:1234/Services/Ping requesting it as is will return XML data.

  • Or one could also invoke it as http://myServer:1234/Services/Ping.xml yielding the same result as above. You have just given it a strong return type given.

  • Or you can invoke it as http://myServer:1234/Services/Ping.json returning JSON data to the caller.
