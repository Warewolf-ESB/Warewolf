------------------- Assign Variable Activity UI Customization  -------------------------------
13. Drag and drop activity in Workflow designer: Assign Variable (can be found under WareWolf group)
14. Click the activity to open Properties window - Small View (to take inputs from user, i.e., user can assign multiple variables)
15. Click Large view (top right side in properties window, to toggle between large/small view)
16. Press Tab key in the last row's New Value column to insert new row in the grid
17. Click Ok/Cancle button to close properties window
